# m3u8をmp4に変換するツール

## 概要
ffmpeg-pythonを用いて，m3u8ファイルをmp4に変換するアプリケーション．

Flask使ってみたかっただけ．

## 使用技術
- フロントエンド
    - HTML
    - CSS
- バックエンド
    - Python (Flask)

## 実行手順
1. [ffmpeg](https://ffmpeg.org/)のインストール
2. Pythonパッケージのインストール
```shell
pip install ffmpeg-python
```
3. 実行してターミナルに出てきたリンクを開く
```shell
python app.py
```
